# 🎬 صانع مقاطع TikTok بالذكاء الاصطناعي (AI TikTok Clips Maker)

مشروع احترافي لتحويل فيديوهات YouTube أو الفيديوهات المرفوعة إلى مقاطع TikTok قصيرة (9:16) باستخدام الذكاء الاصطناعي.

## 🚀 المميزات:
- ✅ **دعم YouTube:** تحميل الفيديوهات مباشرة من روابط YouTube باستخدام `yt-dlp`.
- ✅ **الذكاء الاصطناعي (Gemini AI):** تحليل الفيديو واختيار أفضل اللحظات تلقائياً.
- ✅ **تنسيق TikTok (9:16):** قص الفيديو ذكياً (Smart Crop) ليناسب الهواتف.
- ✅ **التخزين السحابي (ClawCloud):** رفع المقاطع النهائية إلى S3 Storage (ClawCloud).
- ✅ **إدارة الملفات:** حذف الفيديو الأصلي فوراً، وحذف المقاطع من السيرفر بعد 30 دقيقة تلقائياً.

## 🛠️ المتطلبات:
- Python 3.11+
- FFmpeg (مثبت على النظام)
- مفتاح Gemini API
- بيانات الوصول لـ ClawCloud S3 (Access Key, Secret Key)

## 📦 التشغيل المحلي:
1. قم بتثبيت المكتبات:
   ```bash
   pip install -r requirements.txt
   ```
2. قم بتشغيل التطبيق:
   ```bash
   python app.py
   ```

## 🌐 النشر على Render:
المشروع جاهز للنشر بضغطة زر على منصة **Render** بفضل ملف `render.yaml` المرفق. فقط اربط مستودع GitHub الخاص بك بـ Render وسيتكفل هو بالباقي!
